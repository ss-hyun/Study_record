## 소스 코드 삽입
---
~ 와 같은 키보드에 위치한 억음(Grave Accent) 문자 `를 사용하여 다음과 같은 형식

'''(소스코드 형식)  
<코드>    
'''

Ex)  
```python
# python
print("hi")
a = input()
```
```c++
// c++
#include <iostream>
int main(){
    int a = 10;
    return 0;
}
```

# 링크와 이미지 삽입
## 하이퍼링크 (Hyperlink)
---
기본적인 링크 방법으로는 다음과 같은 세 가지 방법이 있다.
* <(링크 주소)>  
* [링크이름](링크주소)  
* [링크이름](링크주소, "부연 설명")  : 부연 설명은 마우스를 갖대다면 보인다.

Ex)  
* <https://www.naver.com>  
* [네이버](https://www.naver.com)  
* [네이버](https://www.naver.com "네이버 바로가기")  
<br>
