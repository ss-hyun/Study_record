이미지 나타내기

이미지를 웹페이지 상에 불러와 표시하기 위해서는 img 태그와 src라는 속성을 사용해준다.
src라는 속성을 이미지 링크 주소 또는 이미지 파일 경로로 설정해주면 이미지를 나타낼 수 있다.
또한 width라는 속성을 사용하면 이미지의 크기를 조정해줄 수 있는데, 숫자나 퍼센테이지를 이용해 속성 값을 설정해준다.

Example
s소스 src="img 링크" width="원래 이미지 크기 대비 원하는 퍼센테이지"
